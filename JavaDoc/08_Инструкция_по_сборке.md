# ИНСТРУКЦИЯ ПО СБОРКЕ ПРОГРАММЫ

## 1. Системные требования для сборки
- Java Development Kit 21
- Apache Maven 3.6+
- 4 ГБ оперативной памяти

## Зависимости проекта
JavaFX 21.0.8 - графический интерфейс
Apache PDFBox 2.0.29 - работа с PDF
BouncyCastle 1.70 - чтение метаданных подписей

## Структура проекта

src/
├── main/
│   ├── java/com/example/
│   │   ├── controller/    # Контроллеры GUI
│   │   ├── model/         # Модели данных  
│   │   ├── util/          # Утилиты
│   │   └── view/          # FXML файлы
│   └── resources/
│       ├── css/           # Стили
│       └── images/        # Графика



## 2. Процесс сборки
```bash
# Клонирование репозитория
git clone [repository-url]

# Переход в директорию проекта  
cd nbdsig

# Сборка проекта
mvn clean package

# Результат: target/nbdsig-1.0-SNAPSHOT.jar
