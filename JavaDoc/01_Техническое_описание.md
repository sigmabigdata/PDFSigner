# ТЕХНИЧЕСКОЕ ОПИСАНИЕ ПРОГРАММЫ
# "Визуализатор штампов электронной подписи"
# Версия 1.0

## 1. Назначение программы
Программа предназначена для визуализации информации об электронных подписях в PDF-документах путем добавления графических штампов и протоколов проверки.

## 2. Основные функции
- Визуализация информации из файлов электронной подписи (.sig)
- Добавление графических штампов в PDF-документы
- Создание протоколов проверки подписей
- Обработка доверенностей в формате XML
- Визуальное размещение элементов на документах

## 3. Технические характеристики
- Язык программирования: Java 21
- Графический интерфейс: JavaFX
- Библиотеки работы с PDF: Apache PDFBox 2.0.29
- Формат входных данных: PDF, SIG, XML
- Формат выходных данных: PDF

## 4. Отсутствие функций криптозащиты
Программа НЕ выполняет:
- Проверку криптографической целостности подписей
- Верификацию сертификатов
- Шифрование/дешифрование данных
- Создание электронных подписей
- Криптографические операции любого вида
