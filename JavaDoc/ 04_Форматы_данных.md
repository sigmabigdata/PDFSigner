# ОПИСАНИЕ ФОРМАТОВ ДАННЫХ

## 1. Входные форматы

### 1.1. PDF документы
- Стандарт: PDF 1.7 и выше
- Кодировка: Base64 не требуется
- Ограничения: без паролей защиты

### 1.2. Файлы подписей (.sig)
- Формат: Производные от PKCS#7
- Содержимое: Метаданные подписи
- Извлекаемая информация:
    - ФИО владельца
    - Организация
    - Должность
    - Сертифицирующий центр

### 1.3. XML доверенности
- Стандарт: XML 1.0
- Кодировка: UTF-8
- Обязательные элементы:
    - СвДов (НомДовер, ДатаВыдДовер, СрокДейст)
    - СвУпПред (ФИО с атрибутами)

## 2. Выходные форматы

### 2.1. Модифицированные PDF
- Добавленные графические элементы
- Текстовые блоки с информацией
- Сохранение исходной структуры документа

### 2.3. Визуальные элементы
- Графические штампы с текстом
- Таблицы с информацией о подписях
- Протоколы проверки
