# РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

## 1. Установка и запуск
### Требования к системе:
- Java Runtime Environment 21 или выше
- 2 ГБ оперативной памяти
- 100 МБ свободного места на диске

### Запуск приложения:
java -jar nbdsig-1.0-SNAPSHOT.jar

## 2. Основные операции

### 2.1. Выбор файлов
- Нажмите "Выбрать файлы"
- Выберите PDF документ и файлы подписей (.sig)
- Программа автоматически распознает информацию о подписантах

### 2.2. Настройка типа документа
- Выберите тип документа из выпадающего списка
- Доступные типы: кредитные договоры, банковские гарантии и др.

### 2.3. Добавление доверенности
- Нажмите "Добавить доверенность"
- Выберите XML файл доверенности
- Программа автоматически извлечет данные

### 2.4. Размещение штампов
- Нажмите "Добавить подписи"
- Укажите страницы для размещения штампов
- Программа добавит визуальные элементы

### 2.5. Создание протокола
- Нажмите "Сформировать протокол"
- Выберите место размещения на документе
- Заполните данные протокола

## 3. Выходные данные
Программа создает модифицированные PDF-файлы с добавленными графическими элементами, содержащими информацию о подписях.
