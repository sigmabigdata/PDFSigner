# ОПИСАНИЕ АЛГОРИТМОВ

## 1. Алгоритм обработки подписей
Вход: Файлы подписей (.sig)
Выход: Структурированная информация о подписантах

Шаги:

Чтение метаданных из файлов .sig
Парсинг информации о владельце подписи
Классификация подписей по категориям:
Подписанты со стороны банка
Подписанты со стороны клиента
Дополнительные подписанты
Формирование графических представлений


## 2. Алгоритм размещения штампов

Вход: PDF документ, информация о подписях, номера страниц
Выход: Модифицированный PDF документ

Шаги:

Анализ структуры PDF документа
Определение областей для размещения штампов
Генерация графических элементов
Добавление элементов на указанные страницы
Сохранение модифицированного документа


## 3. Алгоритм обработки доверенностей

Вход: XML файл доверенности
Выход: Структурированные данные доверенности

Шаги:

Парсинг XML структуры
Извлечение основных атрибутов:
Номер доверенности
Даты действия
ФИО доверенного лица
Валидация формата данных
Интеграция в визуальное представление
