# ЗАЯВЛЕНИЕ О СООТВЕТСТВИИ

Настоящим подтверждаем, что программное обеспечение
"Визуализатор штампов электронной подписи" версии 1.0:

1. Не содержит средств криптографической защиты информации
2. Не выполняет функций шифрования и расшифрования
3. Не осуществляет проверку подлинности электронных подписей
4. Предназначено исключительно для визуализации данных
5. Не обрабатывает информацию ограниченного доступа

Программа использует только открытую информацию из файлов подписей
и не имеет доступа к криптографическим ключам или алгоритмам.
