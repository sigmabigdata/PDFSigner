# МЕРЫ ЗАЩИТЫ ИНФОРМАЦИИ

## 1. Обработка конфиденциальных данных
Программа обрабатывает, но НЕ хранит:
- Персональные данные подписантов
- Реквизиты документов
- Информацию о доверенностях

## 2. Отсутствие криптографических функций
Подтверждается, что программа:
- Не использует алгоритмы шифрования
- Не выполняет проверку ЭП
- Не имеет доступа к закрытым ключам
- Не модифицирует сами подписи

## 3. Временное хранение данных
Все обрабатываемые данные:
- Хранятся только в оперативной памяти
- Удаляются после закрытия приложения
- Не записываются в постоянное хранилище

## 4. Соответствие требованиям
Программа соответствует требованиям для ПО:
- Не содержащего средств криптографической защиты
- Обрабатывающего открытые данные
- Визуализирующего информацию
